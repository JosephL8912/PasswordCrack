# PasswordCrack

### Command Line Arguments

        -b = brute force 

        -d = dictionary attack


        -p = crack plaintext 

        -e = crack encrypted


        -m = MD5
        
        -s = SHA256
        
        -b = bcrypt

### Formating

    - Python3 Cracking.py 81dc9bdb52d04dc20036dbd8313ed055 -b -e -m 
    - Python3 Cracking.py 1234 -d -p
